# Project Title

An Example of Web Service using Laravel Framework 5.4


### Installing

Create a database name 'webservice' using MySQL. Then, execute bellow command to migrate database table:

```
php artisan migrate
```

then, run bellow command:
```
php artisan serve
```

## Running the tests

Open browser and copy this url: http://localhost:8000
